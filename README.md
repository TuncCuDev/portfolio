# Portfolio Website

A modern and responsive portfolio website built with HTML, CSS, and JavaScript. The website showcases my skills, projects, and professional background while providing visitors with an easy way to learn more about my work and get in touch.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Smooth scrolling navigation
* Scroll-based reveal animations
* Interactive project showcase section
* Skills and technology overview
* Contact form with validation
* Active navigation highlighting
* Dedicated Impressum page
* Modern and user-friendly interface

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)
* Responsive Web Design
* Intersection Observer API

## Sections

### Home

Introduction and personal presentation.

### About Me

Overview of my background, experience, and motivation.

### Skills

Presentation of technical skills and technologies.

### Portfolio

Showcase of selected projects and achievements.

### Contact

Contact form and additional contact information.
